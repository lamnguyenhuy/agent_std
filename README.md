# Agent Studio

Agent Studio is a Next.js App Router + shadcn/ui + Tailwind application for reviewing agent behavior changes.

## Adding components

To add components to your app, run the following command:

```bash
pnpm dlx shadcn@latest add button
```

This places UI primitives under `src/components/ui`.

## Using components

To use the components in your app, import them as follows:

```tsx
import { Button } from "@/components/ui/button"
```
